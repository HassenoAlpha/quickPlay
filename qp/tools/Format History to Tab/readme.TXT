Run 'Format Game History to Tab, and Browse to the folder with your game history or info file in it (It says browse to your QuickPlay folder as the default place to store these used to be QP\dats, but you can search for them anywhere). A copy of your existing history named 'history.dat.old' will be made, and a new history.dat will be created.

That's it! Fire up QP, and see the difference.
